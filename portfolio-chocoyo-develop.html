CREDICOM (Nicaragua)

Reto:

Crear un ERP para controlar la operación que va desde el registro de clientes, aprobación de crédito, programación de desembolso, programación de rutas y cobro, todo esto para la parte Web. Adicional necesitaban gestionar todas las rutas de cobros, asignar cobradores y realizar la operación en si a tarves de una App móvil.

Entregables y Tecnología usada:

  • UI/UX
  • ERP Web
  • App móvil de cobro
  • Android (Móvil) y Ruby on Rails (Backend)
  • PostgreSQL

Entregables y Tecnología usada:

  • UI/UX
  • Panel de administración
  • NodeJs
  • SQLServer
VALUE FOR WOMEN (Inglaterra)

Reto:

V4W requería la automatización de una plataforma que le permitiera realizar Test que midieran la equidad de género entre todas las empresas asociadas. De tal manera dicha plataforma tendría que permitir al asociado registrarse, realizar los test, ver test anteriores y visualizar en el tiempo la evolución de este indicador para medir de alguna manera el impacto económico que estas mejoras producen.

Entregables y Tecnología usada:

  • Panel de Administración - Python con Django
  • Plataforma Web - Python con Django
  • Postgres SQL
  • UI/UX
WIMIX (BAC): WIMIX ADS Y WIMIX CONTENTS

Entregables y Tecnología usada:
